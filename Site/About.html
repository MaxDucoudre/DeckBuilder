FAQ

Qui sommes nous ?

Nous sommes un groupe de 4 élèves venant du DUT informatique de Fontainebleau eyant un projet tutoré sur un sujet que nous devions choisir !
Ce projet est un roguelike games sous Unix qui est un type de jeu que nous aimiont tous est qui est notre sujet de projet !

Comment nous contacter ?

Il suffit de cliquer sur la page "Feedback" pour nous envoyer vos retours sur d'eventuel bug ou avis !

Comment télecharger le jeu ?

Il vous suffit d'aller sur l'onglet "Telechargement" et Appuyer sur le bouton telecharger !
